ResQ is a comprehensive desktop application designed to **manage disaster relief operations efficiently**. It enables coordinated efforts between **volunteers, donors, victims, and administrators**, ensuring timely aid and resource distribution.

---

## **Features**

### **User Roles**

* **Admin / Super Admin**

  * Manage users, donors, volunteers, and resources.
  * Allocate aid and assign tasks to volunteers.
* **Volunteers**

  * Register and view nearby tasks (within 5 km radius).
  * Update task status after completing assignments.
* **Donors**

  * Donate resources or funds.
  * Track donations and request receipts.
* **Victims**

  * Request aid and resources.
  * Receive notifications when help is available.

---

### **Core Functionality**

* **Donation Management**: Accept and track donations, both monetary and in-kind.
* **Resource Requests & Allocation**: Match resources to victims‚Äô requests efficiently.
* **Volunteer Assignment**: Assign tasks to volunteers based on location and availability.
* **5 km Radius Task Mapping**: Volunteers can see tasks within a 5 km radius for quick response.
* **Admin Dashboard**: Comprehensive interface to monitor relief operations and generate reports.

---

## **Tech Stack**

* **Frontend**: WinForms (C++)
* **Backend**: SQL Server
* **Database**: SQL for users, donors, volunteers, resources, and tasks
* **Key Features**:

  * Real-time volunteer-task mapping
  * User authentication and role-based access
  * Data-driven reports for admins

---

## **Installation & Setup**

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/ResQ.git
```

2. Open the solution in **Visual Studio Code**.
3. Set up the SQL Server database using the provided schema.
4. Configure database connection strings in the application.
5. Run the project (`F5`) to start the application.
